新手必看！OpenClaw AI 助手零基础安装指南

OpenClaw
OpenClaw
发布日期:   2026-03-11
文章字数:   541

✨ 什么是 OpenClaw？

OpenClaw 是一个强大的 AI 智能体框架，让你能够：

  • 🤖 创建和管理自己的 AI 助手
  • 💬 无缝连接 Telegram、Discord、WhatsApp 等聊天平台
  • 🔍 联网搜索、浏览器操作、文件管理等能力
  • 🎨 自定义技能扩展，打造专属 AI 工作流

📋 准备工作

系统要求

✅ Node.js 22+ 
✅ macOS / Linux / Windows
✅ 至少 4GB 内存（建议 8GB+）

检查版本

node --version  # 应该显示 v22.0.0 或更高
npm --version   # 确认 npm 已安装

🚀 快速安装（3 步搞定！）

Step 1️⃣：一键安装脚本

macOS/Linux 用户：

curl -fsSL https://openclaw.ai/install.sh | bash

Windows 用户：

iwr -useb https://openclaw.ai/install.ps1 | iex

💡 安装过程会自动配置环境变量和依赖，耐心等待完成！

Step 2️⃣：运行向导程序

安装完成后，执行以下命令启动配置向导：

openclaw onboard --install-daemon

此时会出现交互式界面，引导你设置：

  • 🔐 认证配置
  • 🌐 Gateway 服务设置
  • 📱 可选的聊天渠道（Telegram/微信等）

跟随提示一步步操作即可！✅

Step 3️⃣：验证安装成功

检查 Gateway 状态：

openclaw gateway status

打开控制面板：

openclaw dashboard

如果控制界面正常加载，恭喜你！🎉 OpenClaw 已成功运行！

🔧 进阶设置（可选）

前台运行 Gateway（调试用）

openclaw gateway --port 18789

发送测试消息

需要先配置聊天渠道：

openclaw message send \
  --target +15555550123 \
  --message "Hello from OpenClaw"

💡 常用命令速查

命令 功能
openclaw gateway status 检查服务状态
openclaw dashboard 打开控制面板
openclaw message send 发送消息
openclaw help 查看帮助

🌟 接下来做什么？

完成基础安装后，你可以探索：

  • 🔗 连接更多聊天渠道（Telegram、Discord 等）
  • 🛠️ 学习创建自定义技能（skills）
  • 🤖 部署自己的 AI Agent
  • 📚 查看完整文档：https://docs.openclaw.ai

❓ 遇到问题？

  • 🐛 官方 GitHub：https://github.com/openclaw/openclaw
  • 📖 完整文档：https://docs.openclaw.ai
  • 💬 Discord 社区：https://discord.com/invite/clawd

🎉 结语

至此，你已经成功拥有了自己的 AI 智能体框架！OpenClaw 的强大之处还远不止这些，欢迎继续探索更多功能和技能扩展～
